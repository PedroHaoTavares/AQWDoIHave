## Sobre
AqwDoIhave é uma extensão Chromium que suporta Manifest v3, usada para o AdventureQuest Worlds (AQW).

## Instalação

<details><summary><strong>Instalação padrão para navegador</strong></summary>

Siga os passos abaixo para instalar a extensão em qualquer navegador baseado em Chromium:

1. **Baixe a pasta `Extension`** (pode ser clonando o repositório ou baixando o ZIP).
2. Abra a página de extensões do seu navegador:  
   - Chrome: `chrome://extensions`  
   - Brave: `brave://extensions`  
   - OperaGX/Edge/Outros: equivalente em cada navegador.
3. **Ative o “Modo Desenvolvedor”** (canto superior direito).
4. Clique em **“Carregar sem compactação” (Load Unpacked)**.
5. Selecione a pasta **`Extension`** que você baixou.

Pronto! A extensão estará ativa.

</details>

## Uso
  <li>Acesse https://account.aq.com/AQW/Inventory, faça login se necessário, clique em <em>Inventory</em> e aguarde carregar.</li>
  <li>Vá para qualquer página da AQW Wiki dentro do domínio http://aqwwiki.wikidot.com/. Todos os itens que você possui serão destacados.</li>
  <li>Para atualizar, clique no botão <strong>Update Inventory</strong> e aguarde 10 segundos, irá aparecer uma mensagem de que foi atualizado. Você será redirecionado para a página da wiki que estava acessando.</li>
